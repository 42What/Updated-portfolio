Programming Projects

EECU budget calculator: This website was designed to be a budget calculator that factors in taxes, insurance cost and more. Using JavaScript, this website is capable of calculating monthly pay using only the users yearly salary. This is then used to budget payments and displays how much money would be left afterwards. The website was made in VScode and uses HTML, CSS, and JavaScript. It was in this project that I learned how to work with JavaScript. The website was a success in the fact that it was able to do calculations and display the results, however, there were many challenges.This included the sizing of areas and certain parts of the JS. These challenges were overcome by myself and my partner on the project. We collaborated on this project and communicated our ideas and how to solve the problems we had with coding.
